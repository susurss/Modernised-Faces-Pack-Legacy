# Welcome to my Museum Exhibit!

This is an archived version of the Modernised Faces Pack, in case some of you don't like the upcoming revamped style of the pack. If you are looking for the 2.0 beta, you can find it here: https://github.com/susurss/Modernised-Faces-Pack/releases

This will NOT be updated unless it breaks

**THIS PACK WILL NOT WORK WITHOUT HAPPY TEXTURES INSTALLED**

# If Not Using Geode

Find the Version 1.3.1 release and replace the relevant files in Geometry Dash's "Resources" folder

# If Using Geode

Download these mods:
  - "Texture Loader" by Geode Team
  - "Texture Workshop" by Uproxide
  - "Happy Textures x3c" by Alphalaneous

This texture pack is included in the Texture Workshop so you can download it there to save some hassle

# Compatible Mods

  - "More Difficulties" by Uproxide
  - "GodlikeFaces" by AdyaGD & hiimjustin000
  - "Grandpa Demon" by itzKiba (If you're even on a version of the game it works on lol)
